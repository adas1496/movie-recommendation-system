## 📋 Project Overview

This project builds a simple movie recommendation system using the MovieLens dataset. It recommends movies based on user ratings and genre similarity.

**Key Features:**
- Genre-based recommendations
- Collaborative filtering (user similarity)
- Hybrid approach (rating + genre)

---

## 📊 How It Works

- Uses User-Movie rating matrix
- Calculates similarity between movies using Cosine Similarity
- Suggests similar movies based on user's past ratings
